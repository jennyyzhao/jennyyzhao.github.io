#### Sweater Weather Coding Assignment!
In this assignment, I was tasked to code a program that would decide the appropriate weather gear depending on whatever value you put into the temperature integer variable. One tip that I would give to my past self completing this assignment is that if statements don't need semicolons as we are not closing a thought but rather beginning one.
