#### Odd Menu Coding Assignment!
In this assignment, I was tasked to code a program that would decide if the person would get something from the regular menu or the secret menu depending on whatever value you put into the Numberchoice and Mealoption integer variable. One tip that I would give to my past self completing this assignment is to double check what mode of VS Code I'm in. A lot of my could wouldn't compile and I kept on getting "idnumber=1" even though I had no bugs in my code and it was because I was in another mode in VS Code.
