#### Jammin'!
In this assignment, I was tasked to code a program that play a song on the buzzer of my arduino board when the switch was on. I got the code for the song from this website (https://projecthub.arduino.cc/rowan07/rickroll-piezo-buzzer-b3d9fd). One tip for my past self would be to take a closer look at the example code as when I first tried to run it, it didn't work as I hadn't changed the output to the pin that matched my board.
